## 📌 Sobre o Projeto

Nesta atividade, o objetivo não foi escrever código, mas sim desenvolver o raciocínio lógico — a habilidade mais fundamental para qualquer programador.

O desafio consistiu em decompor ações rotineiras em algoritmos precisos, utilizando duas representações clássicas:

- **Pseudocódigo** — descrição estruturada e textual da lógica
- **Fluxograma** — representação visual do fluxo de decisões e ações

Cada algoritmo segue rigorosamente a estrutura:

```
Entrada → Processamento → Saída
```

## 🎨 O Alfabeto dos Desenvolvedores — Símbolos de Fluxograma

Na lógica de programação, cada forma geométrica tem um papel específico no fluxograma:

| Símbolo | Nome | Função |
|---------|------|--------|
| ▱ Trapézio | Entrada Manual | Dados inseridos pelo usuário via teclado ⌨️ |
| ▱ Paralelogramo | Entrada/Saída Genérica | Qualquer dispositivo fornecendo ou recebendo dados |
| 🖊️ Lápis/Bala | Saída em Vídeo | Exibição de dados na tela do computador 🖥️ |
| 📄 Papel Rasgado | Saída em Impressora | Resultado impresso em papel |
| ◇ Losango | Decisão | Ponto de comparação — o algoritmo escolhe o caminho 💎 |
| ⭕ Círculo | Conector | Organiza o diagrama e evita linhas cruzadas |

## 📦 Variáveis e Constantes

> *"Onde guardamos os tesouros do programa."*

- **Constantes** — Valores fixos e imutáveis. Ex: `PI = 3.1415...`
- **Variáveis** — Espaços nomeados na memória RAM para guardar dados temporários.

### Tipos de Dados Suportados

| Tipo | Exemplo |
|------|---------|
| `Inteiro` | `10`, `-3`, `200` |
| `Real` | `3.14`, `-0.5` |
| `Caractere` | `"SENAI"`, `"Olá"` |
| `Lógico` | `Verdadeiro`, `Falso` |

### ⚠️ Regras de Nomenclatura
- ❌ Sem espaços em branco
- ❌ Não pode começar com números
- ❌ Não use palavras reservadas da linguagem

## 🔢 Operadores

### Aritméticos — Hierarquia de Precedência

```
1º  ^        →  Potenciação e Radiciação
2º  * /      →  Multiplicação e Divisão
3º  + -      →  Adição e Subtração

💡 Use parênteses () para forçar a prioridade!
```

### Relacionais — Para comparações e decisões

| Operador | Significado |
|----------|-------------|
| `=`  | Igual a |
| `<>` | Diferente de |
| `>`  | Maior que |
| `<`  | Menor que |
| `>=` | Maior ou igual a |
| `<=` | Menor ou igual a |
